Fork of Nyr's openvpn-install

OpenVPN road warrior installer for Debian-based distros.

This script will let you setup your own VPN server in no more than one minute, even if you haven't used OpenVPN before. It isn't bulletproof but it has been designed to be as unobtrusive and universal as possible.

Prerequisites: If on a VPS, make sure that you submit a ticket to enable TUN module for openvpn.

Installation Run the script and follow the assistant:

Optional prerequisite for cloning git repo: apt-get install git

git clone https://github.com/droidzone/openvpn-install.git bash openvpn-install/openvpn-install.sh

Once it ends, you can run it again to add more users.
